# MidMeet - Tìm chỗ tụ tiện nhất cho nhóm bạn

App giúp cả nhóm tìm được quán cafe/karaoke/bida... ở giữa chỉ trong 30 giây!

### Tech Stack
- Frontend: React Native + Expo
- Backend: Django REST Framework + PostgreSQL + PostGIS
- Maps: Google Maps API
- Realtime: Socket.IO (sắp có)

**Backend:**
```powershell
cd backend
..\venv\Scripts\activate
python manage.py runserver
